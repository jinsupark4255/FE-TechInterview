## 작동 방식
1. 사용자가 브라우저에 도메인 네임(www.example.com)을 입력.
2. 사용자가 입력한 URL 주소 중에서 도메인 네임 부분을 DNS 서버에서 검색, DNS 서버에서 해당 도메인 네임에 해당하는 IP 주소를 찾고 사용자가 입력한 URL 정보와 함께 전달.
